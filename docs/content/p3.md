## 1. Ciclo `for`

El ciclo `for` es uno de los m√°s utilizados en JavaScript. Es ideal cuando conocemos de antemano cu√°ntas veces queremos que se ejecute el bucle.

2. **Condici√≥n**: Se eval√∫a antes de cada iteraci√≥n; si es `true`, contin√∫a el ciclo

## Buenas Pr√°cticas

!!! tip "Consejos para usar ciclos eficientemente"
    
    1. **Evita bucles infinitos**: Siempre aseg√∫rate de que la condici√≥n del ciclo eventualmente se vuelva `false`
    
    2. **Usa nombres descriptivos**: 
       ```javascript

       // ‚úÖ Bueno
       for (let indiceEstudiante = 0; indiceEstudiante < estudiantes.length; indiceEstudiante++) { }
       ```
    
    3. **Minimiza el trabajo dentro del ciclo**:
       ```javascript

       for (let i = 0; i < array.length; i++) { }
       
       // ‚úÖ Bueno - calcula length una sola vez
       for (let i = 0, len = array.length; i < len; i++) { }
       ```
    
    4. **Usa el ciclo apropiado para cada situaci√≥n**
    
    5. **Considera m√©todos de array modernos** cuando sea apropiado:
       ```javascript
       // En lugar de:
       for (let i = 0; i < numeros.length; i++) {
           console.log(numeros[i] * 2);
       }
       
       // Considera:
       numeros.forEach(numero => console.log(numero * 2));
       ```

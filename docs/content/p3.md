## 1. El ciclo `for`

### 1.1 Sintaxis cl√°sica

## 4. Control de flujo dentro de ciclos

| Sentencia      | Efecto                                                  |
|----------------|---------------------------------------------------------|
| `break`        | **Termina** el ciclo de manera inmediata.               |
| `continue`     | **Salta** al siguiente ciclo sin ejecutar el resto.     |
| `return`       | (en funciones) **Sale del ciclo y de la funci√≥n**.      |

!!! example "break y continue"
    ```js

    for (let i = 0; i < 20; i++) {
        if (i % 2 === 0) continue; // salta pares
        if (i > 9) break;          // termina cuando llega a 10
        console.log(i);            // 1 3 5 7 9
    }
    ```

## 7. Etiquetas (labels) ‚Äì uso avanzado
